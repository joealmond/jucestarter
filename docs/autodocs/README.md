# Autodocs Reference

> Documentation about the autodocs template itself — how it works, agent capabilities, and best practices.

These files are part of the **autodocs template** and describe how the AI-powered documentation system works. They are NOT project-specific documentation.

When you add this template to your project:
- **This folder** (`docs/autodocs/`) = autodocs reference docs (read these to learn the system)
- **`docs/know-how/`** = your project's lessons learned (add your own entries here)
- **`docs/architecture/`** = your project's ADRs (add your own entries here)
